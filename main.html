Question-2: What is the difference between innerHTML, innerText, and textContent ?


the difference between innerHTML, innerText, and textContent is :

textContents is all text contained by an element and all its children that are for formatting purposes only. innerText returns all text contained by an element and all its child elements. innerHtml returns all text, including html tags, that is contained by an element.

Question-3: What is event delegation in the DOM?


The Event delegation in the DOM is :

Event delegation is a design pattern in JavaScript used to efficiently manage and handle events on multiple child elements by attaching a single event listener to a common ancestor element.

Question-4: What is event bubbling in the DOM?


Event bubbling in the DOM :

Event bubbling in the Document Object Model is when an event is triggered on an element and then propagates up to its parent elements.
